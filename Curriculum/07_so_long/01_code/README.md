## Goal:
Collect all collectibles while avoiding collisions with enemies, then cross the finish line and collect the prize too.

## Keyboard
- Exit:

    <kbd>Esc</kbd>
- Start/Stop:
    
    <kbd>Space</kbd>
- Acceleration:

    <kbd>W</kbd> / <kbd>↑</kbd>
- Break:

    <kbd>S</kbd> / <kbd>↓</kbd>
- Left:

    <kbd>A</kbd> / <kbd>←</kbd>
- Right:

    <kbd>D</kbd> / <kbd>→</kbd>
- Honk:

    <kbd>H</kbd>
